Contribute

nextECA is built by the community. Help us grow — add opportunities, fix bugs, improve designs, or spread the word.

How to Contribute

Step 01

Fork the Repository

Go to our GitHub page and fork the nextECA repository to your account. This gives you your own copy to work on.

Step 02

Add or Edit Data

Open js/data.js and add a new opportunity entry following the existing format. Fill all required fields.

Step 03

Submit a Pull Request

Once your changes look good, open a Pull Request with a clear description. Our team reviews and merges within 48 hours.

Step 04

Get Credited

All contributors are credited in our README.md and Contributors page. Your name lives in the project history forever.

Adding an Opportunity — Format

// Add this object to the opportunities array in js/data.js
{
  id: 101, // Next sequential number
  title: "Your Opportunity Title",
  type: "hackathon", // hackathon | olympiad | competition | scholarship | fellowship | volunteering
  country: "🇧🇩 Bangladesh", // Flag emoji + country name
  description: "Short description under 200 characters.",
  deadline: "2025-06-30", // YYYY-MM-DD format
  link: "https://official-registration-url.com",
  urgent: false // true if deadline is within 7 days
}

Other Ways to Help

🐛

Report Bugs

Found something broken? Open a GitHub Issue with details — browser, device, what went wrong, and screenshots if possible.

🎨

Design & UI

Have a design improvement in mind? We welcome CSS contributions, new page templates, or accessibility fixes.

✍️

Write Content

Help us write detailed guides for specific ECA categories, country profiles, or university ECA culture pages.

📣

Spread the Word

Share nextECA with your school, university, or online community. Every share helps a student find an opportunity.
